# mindset-map.md — Глобальная карта mindset-сдвигов

Этот файл отслеживает attitude-изменения студентов **через все чанки курса**.  
Обновляется по мере прохождения сессий и фиксации рефлексий.

Структура: каждый mindset-сдвиг — это переход **от → к**.  
Статусы: `seeded` (посеяно), `emerging` (проявляется), `internalized` (усвоено).

## Как обновлять этот файл

1. После каждой сессии: обновить статусы в таблице для студентов, прошедших рефлексию.
2. Статусы:
   - `seeded` → тема поднята, студент слышал концепцию
   - `emerging` → студент начал применять в своём контексте
   - `internalized` → студент может объяснить, защитить и учит других
   - `—` → чанк ещё не пройден
3. При появлении нового чанка с attitude-компонентом: добавить строку в таблицу.
