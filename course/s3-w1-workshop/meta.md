## Проверка усвоения

### Уровень 1-2 (Помнит / Понимает)
1. Что такое техника "5 вопросов перед кодом"? Зачем она нужна?
2. Чем HTML-дашборд лучше PowerPoint презентации для передачи информации клиенту?
3. Как использовать префиксы в названиях папок при работе с агентами?

### Уровень 6 (Создаёт) — практическое задание
Создай HTML-дашборд под одну из своих реальных бизнес-задач:
1. Используй технику "5 вопросов" для формулировки содержимого
2. Попроси агента создать HTML-файл с Tailwind CSS
3. Открой в браузере, проверь, что выглядит разумно
4. Зафиксируй: что получилось, что поменял бы

## Definition of Done

**Минимальный DoD** (Блум: уровень 3):
- [ ] Техника "5 вопросов" применена хотя бы раз
- [ ] HTML-файл создан и открывается в браузере

**Полный DoD** (Блум: уровень 5-6):
- [ ] Дашборд содержит реальные данные или структуру из своего контекста
- [ ] Понимает, как обновить дашборд при изменении данных
- [ ] Зафиксировал рефлексию: что изменил по сравнению с первым промтом
