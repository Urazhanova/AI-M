## Проверка усвоения

### Знание (Knowledge)
- Какие три основные области изменились в компании-кейсе после принятия стратегического решения о AI-пивоте?
- Перечислите 3-5 примеров типов корпоративной информации, которые могут быть интегрированы в "Shared Workspace" (Второй Мозг Компании).

### Понимание (Comprehension)
- Объясните, почему концепция "Второго Мозга Компании" является фундаментальной для AI-native организации.
- Как изменилось понимание спикера относительно "специальных AI-разработчиков" в процессе формирования команды?

### Применение (Application)
- Предложите, какие два новых источника данных из вашей текущей работы вы бы интегрировали в "Shared Workspace" для создания нового AI-агента.
- Опишите гипотетический сценарий использования AI-агента для рутинной задачи в вашей компании, основываясь на принципах "Второго Мозга".

### Анализ (Analysis)
- Проанализируйте, какие основные вызовы и возможности связаны с переходом B2B SaaS компании к AI-native модели.
- Сравните и противопоставьте традиционные системы управления знаниями с концепцией "Второго Мозга Компании" с точки зрения их эффективности для AI-агентов.

### Синтез (Synthesis)
- Разработайте упрощенную структуру папок для "Shared Workspace" в контексте функционального отдела (например, HR или Маркетинг) вашей компании.
- Предложите, как можно было бы начать процесс вовлечения руководства вашей компании в стратегический AI-пивот, используя опыт спикера.

### Оценка (Evaluation)
- Оцените потенциальные риски (например, безопасность данных, сопротивление сотрудников) при внедрении "Второго Мозга Компании" в крупной организации.
- Какие метрики вы бы использовали для оценки успешности стратегического AI-пивота и внедрения "Второго Мозга Компании" в вашей организации?

## Definition of Done

### Минимальный DoD (Bloom 1-3)
- Студент может определить основные этапы и изменения, произошедшие в компании в процессе AI-пивота.
- Студент может перечислить ключевые источники информации, формирующие "Второй Мозг Компании".
- Студент может предложить конкретные примеры использования AI-агентов, работающих с данными из "Второго Мозга".

### Полный DoD (Bloom 4-6)
- Студент может проанализировать стратегические преимущества и потенциальные препятствия для внедрения "Второго Мозга Компании".
- Студент может предложить структурированный план по адаптации концепции "Shared Workspace" к реалиям конкретного подразделения или компании.
- Студент может оценить подходы к управлению изменениями и вовлечению стейкхолдеров при переходе к AI-native модели.
