## Проверка усвоения

### Знание (Knowledge)
- Перечислите две основные выгоды от внедрения AI-агентов в команду разработки.
- Какие два основных типа внешних систем данных упоминаются как важные для AI-агентов?

### Понимание (Comprehension)
- Объясните, почему страх увольнения сотрудников при внедрении AI-агентов считается необоснованным.
- Приведите примеры вдохновляющих мероприятий, которые могут повысить мотивацию сотрудников к освоению AI-инструментов.

### Применение (Application)
- Опишите, как AI-агент может использовать "источник правды" для обработки данных инвестиционных сделок, если он хранится в Asana.
- Предложите, как руководство может использовать выделение рабочего времени для экспериментов с AI для снижения восприятия новой задачи как дополнительной нагрузки.

### Анализ (Analysis)
- Сравните подход "синхронизировать все данные локально" и подход "агент подключается к системам" с точки зрения их эффективности и целесообразности для крупной компании.
- Проанализируйте, как изменение характера работы инженеров (меньше написания кода, больше спецификаций и тестирования) влияет на их профессиональное развитие.

### Синтез (Synthesis)
- Предложите, какой новый функционал для HR-команды мог бы создать AI-агент, помимо подготовки к интервью, используя концепции управления данными из лекции.
- Разработайте минимальный набор шагов для интеграции AI-агента, работающего с данными клиентов, в существующую CRM-систему компании.

### Оценка (Evaluation)
- Оцените, является ли поэтапное внедрение AI-аагента (как в примере с фармацевтической компанией) оптимальной стратегией для снижения рисков. Обоснуйте.
- Оцените, насколько эффективно проведение внутренних хакатонов может повысить мотивацию сотрудников к освоению новых AI-инструментов.

## Definition of Done

### Минимальный DoD (Bloom 1-3)
- Студент способен перечислить как минимум 2 выгоды и 2 метода мотивации при внедрении AI-агентов, а также объяснить принцип "источника правды".
- Студент может описать типовой рабочий процесс AI-агента с данными, включая этапы извлечения, обработки и сохранения информации.

### Полный DoD (Bloom 4-6)
- Студент разрабатывает стратегию преодоления сопротивления сотрудников внедрению AI-агентов, учитывая различные страхи и методы мотивации.
- Студент проектирует упрощенную схему взаимодействия AI-агента с двумя внешними системами данных, определяя "источники правды" и логику потока данных.
