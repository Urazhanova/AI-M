## Проверка усвоения

### Уровень 1-2 (Помнит / Понимает) — вопросы
1. Назови 5 компонентов формализации задачи.
2. В чём разница между контентом и контекстом?
3. Почему загружать весь массив данных в модель — плохая идея?
4. Что такое эффект Гудхарта в контексте AI-метрик?

### Уровень 3 (Применяет) — практическое задание
Возьми одну конкретную рутинную задачу из своей работы. Заполни шаблон:
```
Задача: [что нужно сделать]
Источники контекста: [откуда берётся информация]
Правила обработки: [5-7 конкретных ограничений]
Формат результата: [что должно быть на выходе]
Критерий качества: [как понять, что результат хороший]
```
Запусти агента с этим контекстом. Зафиксируй результат.

### Уровень 4-5 (Анализирует / Оценивает) — рефлексия
- Где агент справился хуже, чем ты ожидал? Почему?
- Что ты поменял бы в правилах или контексте при следующей итерации?
- Как бы ты измерял улучшение результата на следующей попытке?

## Definition of Done

**Минимальный DoD** (Блум: уровень 3):
- [ ] Задача формализована по 5 компонентам (письменно)
- [ ] Агент запущен с этим контекстом хотя бы один раз

**Полный DoD** (Блум: уровень 4-5):
- [ ] Все 4 вопроса Knowledge отвечены без подсказок
- [ ] Карта источников контекста заполнена
- [ ] Сделана хотя бы одна итерация улучшения правил по результату агента
- [ ] Разница "было / стало" зафиксирована в progress.md
